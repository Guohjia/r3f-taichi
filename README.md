## 使用github登录
1、git clone git@github.com:Guohjia/r3f-taichi.git

2、安装依赖，启动node server
```shell
$ git checkout feature/main
$ npm i
$ npm node src/server/node.js
```

3、启动项目，访问链接，点击登录按钮测试
```shell
$ npm start
$ open http://localhost:8000/r3f-taichi/login
```
